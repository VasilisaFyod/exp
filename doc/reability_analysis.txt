Для оценки надёжности разработанного приложения были применены статические и динамические методы.

Статические методы использовались для анализа исходного кода без его выполнения. В ходе анализа были выявлены запахи кода, такие как дублирование логики и отсутствие комментариев, что позволило оценить качество и поддерживаемость программы.

Динамические методы заключаются в выполнении программы и проверке её корректности. Для этого был разработан модульный тест с использованием xUnit, а его автоматический запуск реализован с помощью GitHub Actions.
